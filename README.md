# Information on Nuclear Energy in Singapore

Interactive educational tools providing balanced, evidence-based information on nuclear energy in Singapore&rsquo;s context.

## Interactive Apps

### [Energy Scenario Explorer](09_interactive_app/)

Model nuclear energy scenarios for Singapore &mdash; explore how different reactor types, deployment scales, and timelines affect energy output, land use, carbon emissions, and waste.

- Compare small modular reactors (SMRs) and advanced reactor designs
- Visualise land footprint against Singapore landmarks
- See carbon reduction and waste projections
- Built with vanilla JS and Chart.js

### [Nuclear Safety Explorer](10_safety_education/)

Learn about nuclear safety through interactive visualisations covering six topics:

- **Radioactivity** &mdash; what radiation is and everyday exposure levels
- **Defence in Depth** &mdash; the multiple barrier approach to reactor safety
- **Safety Concepts** &mdash; passive safety, containment, and emergency systems
- **Historical Accidents** &mdash; what happened at Chernobyl and Fukushima, and what changed
- **Advanced Reactors** &mdash; next-generation safety features
- **Fusion Safety** &mdash; how fusion differs from fission in safety profile

## Running Locally

No build step required. Open directly in a browser:

```bash
open 09_interactive_app/index.html
open 10_safety_education/index.html
```

## Live Site

Visit the deployed version at **https://joelfirenze.github.io/info-on-nuclear-energy-sg/**

## Tech Stack

- Vanilla HTML, CSS, and JavaScript
- [Chart.js](https://www.chartjs.org/) for data visualisation (CDN)
- [Inter](https://fonts.google.com/specimen/Inter) typeface (Google Fonts CDN)
- No frameworks, no build tools, no dependencies to install

## Disclaimer

This project provides educational information and does not advocate for or against nuclear energy. All scientific data is sourced from IAEA, NRC, and peer-reviewed literature.
